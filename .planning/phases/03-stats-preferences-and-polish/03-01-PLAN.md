  <action>
    **Create `src/stats.js`** with two exported functions:

    1. `buildDayObjects(year, month, days, weekendsEnabled)`:
       - Import `daysInMonth` from `./dates.js` and `STATUS` from `./storage.js`
       - Loop from d=1 to daysInMonth(year, month)
       - For each day: `const date = new Date(year, month - 1, d)` (numeric constructor per NFR-4)
       - `isWeekend = date.getDay() === 0 || date.getDay() === 6`
       - Capture `today = new Date()` ONCE before the loop (same pattern as calendar.js)
       - `isCurrentMonth = (year === today.getFullYear() && month === today.getMonth() + 1)`
       - `isFuture = isCurrentMonth && d > today.getDate()`
       - `status = days[String(d)] || STATUS.UNSET`
       - Return array of `{ status, isWeekend, isFuture }`

    2. `renderStats(avgEl, totalsEl, state, weekendsEnabled)`:
       - Import `calcAverage` from `./calc.js`
       - Call `buildDayObjects(state.year, state.month, state.days, weekendsEnabled)` to get dayObjs
       - Call `calcAverage(dayObjs, weekendsEnabled)` to get avg (number or null)
       - Set `avgEl.textContent = avg === null ? 'Average: \u2014' : 'Average: ' + avg.toFixed(1) + ' days/week'`
       - Count per-status totals: initialize `counts = { [STATUS.IN_OFFICE]: 0, [STATUS.AT_HOME]: 0, [STATUS.TIME_OFF]: 0, [STATUS.WFA]: 0 }`
       - Loop dayObjs: skip if `d.isFuture`, skip if `d.isWeekend && !weekendsEnabled`, skip if `d.status === STATUS.UNSET`, then increment `counts[d.status]` if it exists in counts
       - Build totals HTML string with colored dots using `innerHTML` on totalsEl:
         ```
         totalsEl.innerHTML =
           '<span class="stat-dot" data-status="in-office"></span> ' + counts[STATUS.IN_OFFICE] + ' In Office' +
           ' · <span class="stat-dot" data-status="at-home"></span> ' + counts[STATUS.AT_HOME] + ' At Home' +
           ' · <span class="stat-dot" data-status="time-off"></span> ' + counts[STATUS.TIME_OFF] + ' Time Off' +
           ' · <span class="stat-dot" data-status="wfa"></span> ' + counts[STATUS.WFA] + ' WFA';
         ```
